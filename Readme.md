## 데일리 CS 퀴즈 채널 알림봇
Discord 데일리 CS 퀴즈 채널 멤버들을 위한 알림봇입니다.

### [구현 기능] 
**1) 시간 알림**
- 매일 7시 반 퀴즈 출제 알림
- 주말 및 지정한 공휴일에 동작하지 않도록 설정 (2024년 공휴일 반영 예정)

**2) 동작 확인**
채팅창에 "RUTHERE"이라고 입력하면 봇이 응답해 줍니다.
